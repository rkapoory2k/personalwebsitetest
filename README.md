# Personal Website

A minimal, Apple-inspired personal website built with HTML, CSS, and JavaScript.

## Features

- **Hero Section** — Clean, centered introduction
- **About Section** — Brief bio and skills
- **Projects Section** — Showcase of work
- **Contact Form** — With client-side validation
- **Responsive Design** — Works on all devices

## Tech Stack

- HTML5
- CSS3 (Custom, no frameworks)
- Vanilla JavaScript

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser

That's it. No build step required.

## File Structure

```
/
├── index.html      # Main HTML file
├── styles.css      # All styles
├── main.js         # JavaScript functionality
├── README.md       # This file
├── scope.md        # Project scope document
├── mvp.md          # MVP definition
└── design.md       # Design specifications
```

## Customization

1. Edit `index.html` to update content (name, bio, projects, links)
2. Modify CSS variables in `styles.css` to change colors
3. Update social links in the footer

## Design Philosophy

Less is more. Clean. Quiet. Confident.

- Pure white background
- Near-black text
- Generous whitespace
- Subtle interactions
- System fonts for speed
