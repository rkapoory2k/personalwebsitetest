# Personal Website - Design

## Design Philosophy

**Less is more.** Clean. Quiet. Confident.

Inspired by Apple's approach: generous white space, sharp typography, and letting content breathe.

---

## Color Palette

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   ██████  Background      #ffffff  (Pure White)             │
```

## Typography

```
Font:      "SF Pro Display" / "Inter" / system-ui
Weights:   400 (body), 600 (semibold), 700 (bold)

Sizes:
  - Hero headline:     72px / 80px
  - Section titles:    48px
  - Body:              17px / 21px
  - Small/caption:     14px

Line height:           1.5 (body), 1.1 (headlines)
Letter spacing:        -0.02em (headlines), normal (body)
```

## Animation Notes

```
Transitions:
  - All hover states:  0.2s ease
  - Page sections:     fade in on scroll (subtle, 0.4s)
  - Mobile menu:       fade 0.3s

Rules:
  - No bouncing
  - No sliding
  - No glows
  - Just opacity and color changes
  - Smooth, imperceptible
```

## Summary

| Element | Style |
|---------|-------|
| Background | Pure white `#ffffff` |
| Text | Near black `#1d1d1f` |
| Accent | Apple blue `#0071e3` (used sparingly) |
| Typography | System font, clean sans-serif |
| Layout | Centered, narrow content, massive whitespace |
| Interactions | Subtle opacity changes only |
| Vibe | Quiet confidence. Let the work speak. |

---

## The Rules

1. When in doubt, remove it
2. White space is not empty — it's breathing room
3. One font, few weights
4. Color is a whisper, not a shout
5. Every element earns its place
